Description:

Konsole plugin for Qt applications Konsole is a terminal emulator built on the KDE Platform. It can contain multiple terminal sessions inside one window using detachable tabs. . Konsole supports powerful terminal features, such as customizable schemes, saved sessions, and output monitoring. . This package provides the kpart needed by other apps to embed konsole in them.
